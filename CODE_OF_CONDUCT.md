# Code of Conduct

Code Orange Dev School is a learning community. We're here to help regular developers and curious Bitcoiners become open-source contributors. That only works if everyone feels welcome.

## Our standards

**Do:**
- Be kind, patient, and encouraging — especially to beginners. Everyone started somewhere.
- Assume good faith. Ask questions before assuming the worst.
- Give constructive, specific feedback. Critique the code, not the person.
- Respect that people contribute across many timezones, languages, and skill levels.
- Help others learn rather than doing the work for them.

**Don't:**
- Harass, demean, or discriminate against anyone for any reason.
- Use sexualized, violent, or hateful language or imagery.
- Dox, share private information, or post others' personal data without consent.
- Spam, scam, shill tokens, or use the community to push unrelated commercial agendas.

## Scope

This applies to all Code Orange spaces — GitHub repos, Discord, workshops (online and in person), and any space where you represent the community.

## Enforcement

If someone is making the community worse, tell us. Contact a maintainer on [Discord](https://discord.gg/xd6dmPF9bA) or email **keypleb@blink.sv**. Reports are handled confidentially. Maintainers may warn, mute, or remove anyone whose behavior is harmful, at their discretion.

## Spirit

We build the community. They build Bitcoin. Be the kind of person who makes others want to keep contributing.
